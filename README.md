# Streamlit AutoML System

## Overview
This project is a Streamlit-based AutoML platform that automates the machine learning pipeline including EDA, model comparison, tuning, and prediction.

## Features
- Automated EDA
- AutoML using PyCaret
- Model comparison
- Hyperparameter tuning
- Explainability
- Clustering
- Prediction

## How to Run
pip install -r requirements.txt  
streamlit run app.py
